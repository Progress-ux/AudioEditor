# TUI MP3 Tag Editor 🎵

Удобный, минималистичный консольный интерфейс (TUI) для редактирования метаданных и ID3-тегов ваших MP3-файлов. Отличное решение для тех, кто предпочитает управлять своей медиатекой прямо из терминала без использования тяжелых графических утилит.

Проект отлично дополняет скрипт [DownloadTrack](https://github.com/Progress-ux/DownloadTracks.git), позволяя быстро приводить в порядок теги скачанных из сети треков.

---

## ✨ Особенности

* **Быстрый файловый менеджер:** Автоматическое сканирование рабочей директории и вывод списка доступных `.mp3` аудиофайлов.
* **Полноценное редактирование:** Изменение полей: *Title, Artist, Album, Genre, Year (Date), Track number, Comment*.
* **Vim-inspired управление:** Навигация спроектирована специально для эффективной работы без использования мыши.
* **Современный стек:** Высокая производительность благодаря компиляции на C++20 и использованию эффективной библиотеки TagLib.

---

## 🛠 Стек технологий

* **Язык:** C++20
* **Интерфейс:** [FTXUI v7.0.0](https://github.com/ArthurSonzogni/FTXUI) (Продвинутая C++ библиотека для построения текстовых UI)
* **Работа с метаданными:** [TagLib](https://taglib.org/)

---

## ⌨️ Управление (Hotkeys)

| Клавиша | Действие |
| :--- | :--- |
| `j` / `k` или `↑` / `↓` | Перемещение вверх / вниз по списку файлов |
| `o` | Открыть редактор метаданных (Edit mode) выбранного файла |
| `q` | Выйти из программы |

## 🚀 Сборка и установка

### Вариант 1: Быстрая установка

Для сборки вам понадобятся компилятор с поддержкой C++20, `CMake` и установленная библиотека `TagLib`.

#### Использование готового пакета (Arch Linux)

Если вы используете Arch Linux, вы можете собрать и установить проект одной командой (все зависимости и `ffmpeg` установятся автоматически):

```bash
makepkg -si
```
Для запуска используйте `editor` в текущей папке

### Вариант 2: Ручная сборка из исходников

**Установка зависимостей (Arch Linux):**
```bash
sudo pacman -S taglib cmake make gcc
```

### Сборка проекта
```bash
git clone https://github.com/Progress-ux/AudioEditor.git 
cd AudioEditor
mkdir build && cd build
cmake ..
make
./editor
```
